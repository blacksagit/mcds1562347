Emmet Examples

Anidar Elementos ( > )

ul>li>a>img 
<ul>
	<li>
		<a href="">
			<img src="" alt="">
		</a>
	</li>
</ul>

Elementos Hermanos ( + )

section+header+nav 
<section></section>
<header></header>
<nav></nav>

Repetir Elementos ( * )

ul>li*3 
<ul>
	<li></li>
	<li></li>
	<li></li>
</ul>

Atributo Clase ( . )

div.contenedor - .contenedor2 
<div class="contenedor"></div>

<div class="contenedor2"></div>

Multiples Clases ( . . . )

div.clase1.clase2.clasen 
<div class="clase1 clase2 clasen"></div>

Atributo Identificador ( # )

nav#menu 
<nav id="menu"></nav>

Adicionar Texto ( { } )

div{Lorem Ipsum Dolor} - a{Click Aquí} 
<div>Lorem Ipsum Dolor</div>

<a href=“”>Lorem Ipsum Dolor</a>

Personalizar Atributos ( [ ] )

a[href=www.google.com] 
<a href="www.google.com"></a>

Numeros Consecutivos ( # )

ul>li.num$*3 - section>article.sec$$$*3 
<ul>
	<li class="num1"></li>
	<li class="num2"></li>
	<li class="num3"></li>
</ul>

Dirección Números Consecutivos ( @- )

ul>li.num$@-*3 
<ul>
	<li class="num3"></li>
	<li class="num2"></li>
	<li class="num1"></li>
</ul>

Base Números Consecutivos ( @n )

ul>li.num$@3*4 
<ul>
	<li class="num3"></li>
	<li class="num4"></li>
	<li class="num5"></li>
	<li class="num6"></li>
</ul>

Subir Nivel ( ^ )

section>article>div^p 
<section>
	<article>
		<div></div>
	</article>
	<p></p>
</section>

Agrupar Elementos ( )

(header>nav)(section>article) 
<header>
	<nav></nav>
</header>
<section>
	<article></article>
</section>
